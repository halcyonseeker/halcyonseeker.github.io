Rip .aax files from Audible with ffmpeg

This tutorial teaches you how to free your audiobooks from Audible's DRM.

My intention is that it be used for personal use, please do not be a parasite

The Boring Bit

Okay, so this is the first tutorial I've ever written, so please bear with me. For many years now I've been an avid listener of audiobooks, and Audible has been my main source of content. But as I learned more tech, I grew disatisfied with my reliance on an Audible app to listen to the audiobooks I PURCHASED. So this tutorial will teach you how to convert the audiobooks you bought from Audible and listen to them anywhere, anywhen, and anyhow.

How (I Think) It Works

In order to listen to your audiobook offline you'll need to find your so-called "activation-bytes". Each audiobook you purchase from Audible contains your imbedded activation-bytes. In order to get them, you need to extract a hash from the audiobook and then decrypt it. With the activation-bytes in hand you can then proceed to decrypt your audiobook into a format that can be played anywhere.

What You Need

  • A basic grasp of the Linux/Unix command line (I dunno if this works on Windows). You need to know how to enter a directory, run an executable file, and stuff.
  • ffmpeg, a wonderful command line utility for manipulation audio and video streams.
  • inAudible-NG's Rainbow Tables Repository on Github. Rainbow tables are a really cool tool for decrypting data. Basically, when brute-forcing encrypted information, there is (as with many other things in computing) a trade-off between memory and cpu usage. Rainbow Tables contain pre-computed values, saving processing time and resources while brute-forcing encrypted data.

How To Do It

If you're lazy like me and just want to listen to your audiobooks how you like, then use this script I wrote. If you want to do things your own way, follow the description below.

Enter the directory containing the cloned tables repository and the .aax files

cd ~/path/to/directory/

Extract the checksum from the .aax file with ffprobe

ffprobe file.aax | grep checksum | awk '{ print $NF }'

Enter the tables repository

cd tables

Extract the activation bytes from the checksum

./rcrack . -h CHECKSUM_FROM_FFPROBE

Convert the .aax file to a .flac with ffmpeg

ffmpeg -activation_bytes BYTE_FROM_TABLES -i file.aax file.flac

And now you should have your audiobook as a DRM free flac file that you can listen to almost wherever you want. Because flac is an inherently free codec, some older proprietary players like the iPod don't support it. This problem can be solved by flashing the RockBox firmware, or converting the .aax file to a .mp3.
